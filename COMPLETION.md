## 🎮 Как это работает

### Поток игры (за 60 секунд)

```
1. Создатель → "Создать игру" → получает 6-значный КОД
2. Остальные → "Войти по коду" → вводят КОД
3. Когда 2-8 человек → создатель кликает "Начать"
4. Каждый ход:
   - Игрок кликает "Бросить кубик"
   - Приземляется на клетку
   - Видит карточку с выбором пути
   - Выбирает A/B/C
   - Остальные могут оспорить (12 сек)
   - Голосуют (25 сек если оспорили)
   - Результат → Влияние + Город изменились
5. После 7 ходов каждого → подсчёт очков
6. Город ≥8? → бонус к результату → ПОБЕДИТЕЛЬ!
```

### Технический стек

```
Frontend:      React 18 + Vite
Database:      Firebase Realtime DB (бесплатно)
Hosting:       Vercel (бесплатно)
Telegram:      Web App API
Authentication: Telegram User ID
```
